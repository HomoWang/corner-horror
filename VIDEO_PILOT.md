# 407 號房：互動影片試片

這份規格把目前的即時場景改造成電影式分支影片。第一版只驗證一件事：玩家能否在不低頭操作手機的情況下，看完一段有節奏的恐怖戲、做一次選擇，並得到兩個連續且可信的結果。

## 試片範圍

- 總長：共通段落約 28 秒，分支各 12–16 秒。
- 畫面：16:9、24 fps、1080p，所有鏡頭維持同一房間、演員與服裝。
- 操作：選項只顯示在大螢幕；玩家將手機／手電筒指向左或右，再按手機中央按鈕確認。
- 聲音：影片保留房間底噪與角色現場聲；電話鈴、低頻配樂、撞擊與尖叫由網頁音軌同步播放，避免每次生成的聲音不一致。
- 字幕：沿用現有主螢幕字幕與手機語音，不把文字燒進影片。

## 故事節拍

### P01 — 來電（8 秒）

房間完全沒有人。鏡頭幾乎靜止，只做極慢推進。第 2.2 秒老電話第一次響，第 4.8 秒第二次響；第二次響起時，中央全家福的玻璃反射像有人經過，但不能看清人形。

旁白／字幕：「凌晨三點十七分。這支電話不該還有線。」

### P02 — 靠近（8 秒）

第一人稱鏡頭從房門方向緩慢移向電話。鈴聲突然中斷，聽筒仍在原位。鏡頭最後停在電話與兩張小照片的穩定近景；電話裡傳出女人極近的呼吸，接著低聲說：「別看門。」轉向右側房門的動作改由 P03 承接，避免單段八秒塞入兩次鏡位變化。

### P03 — 門後（8 秒）

鏡頭從電話慢慢轉向門口。第 4.1 秒門內傳來指甲刮木頭的聲音；約第 6.85 秒，一名濕髮女人突然從門後探出完整人臉並盯住鏡頭。這不是原提示要求的短暫眼睛，但臉部穩定、出現時機有效，因此保留並把它改成限時威脅選擇。

畫面停在女人探出門口的乾淨畫格。左側顯示「立刻甩門」，右側顯示「僵住不動」。選擇倒數 8 秒；玩家只需把手機指向左或右，再按中央鍵確認。

### P04A — 立刻甩門（16 秒，兩段）

第一段由 P03 最後一格延伸：主角抓住房門將它猛力甩上，女人及時縮回黑暗，門板浮現連續抓痕；約第 6.5 秒，門在沒有人碰觸的情況下又自行打開，最後停在空無一人的黑色門縫。第二段以 0.2 秒黑場掩護生成接點的門位差，亮起時門竟已重新關上；它再度開啟，鏡頭轉向電話取得短暫安全感，女人隨即從畫面左下撲近。第 7.1 秒同步長震、尖叫與低頻撞擊。

### P04B — 僵住不動（16 秒，兩段）

第一段由 P03 最後一格延伸：主角完全不動，女人以不自然地緩慢方式多探出一些身體，但始終不跨過門檻，低聲說「很好。不要眨眼」。第二段讓電話在身後響起；如果鏡頭分心，女人便在下一個視線死角消失。她最後貼近左耳說「現在，換我看著你」，畫面停黑並留下兩次短震。這條分支保留後續生還路線，不重複同一個跳臉。

## 影片生成策略

不要生成一支 40 秒影片。每個節點獨立生成，再用最後一格延伸下一段：

1. P01 使用 `room407-master.png` 作為第一格。
2. P02 使用 P01 的最後一格作為第一格，並把房間主圖當場景參考。
3. P03 延伸 P02，人物身分圖只作 Ingredient／Reference，不允許完整人物提前出現。
4. P04A 同時引用房間主圖、人物身分圖、異常表演圖與跳臉圖；跳臉圖只作最後狀態參考。
5. P04B 引用房間主圖、人物身分圖與異常表演圖，不引用跳臉圖。
6. 每段至少生成三個版本，只比較動作與連續性；不要因單張畫面漂亮就選擇動作崩壞的版本。

## 驗收標準

- 不暫停逐格觀看時，觀眾不能察覺家具、門框、照片或演員長相跳變。
- P01 前 6 秒沒有鬼臉或明顯人影，恐怖感先由電話與反射建立。
- P03 的完整人臉約在最後 1.1 秒探出，臉部與門框不可漂移；選擇畫面直接凍結在最後一格。
- P04A 的最後撲近必須在黑場後 0.2 秒內進入，尖叫不可提前。
- 兩個分支的第一格必須與選擇畫面完全銜接，不能有明顯跳切或重新構圖。
- 手機從選擇出現到確認全程只需要中央按鈕，不增加手機文字選單。
